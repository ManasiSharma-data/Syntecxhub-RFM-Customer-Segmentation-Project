# 📊Syntecxhub RFM Customer Segmentation Project

## 📌 Project Overview
This project analyzes customer behavior using RFM (Recency, Frequency, Monetary) analysis to segment customers and provide business insights.

## 🎯 Objectives
- Identify high-value customers
- Segment customers based on RFM scores
- Analyze customer behavior patterns
- Improve marketing strategies

## 📊 Dashboard Features
- Total Customers, Revenue (₹ Cr), Frequency & Recency KPIs
- Customer Segmentation (Champions, At Risk, Loyal, etc.)
- RFM Score Trends
- Revenue Contribution by Segment
- Top Customers Table

## 🛠️ Tools Used
- Excel
- Power BI
- DAX
- Data Cleaning & Transformation

## 🚀 Key Insights
- Champions contribute highest revenue
- At-risk customers need attention
- Loyal customers drive repeat purchases
